GNN-based Method for Tractography Segmentation

This project focuses on developing a novel method for white matter tractography segmentation using Graph Neural Networks (GNNs) and supervised contrastive learning. The method significantly improves the accuracy of segmenting white matter fibers in brain imaging.

Key Features:

  • Representation of tractography data as a graph structure
  • Novel supervised contrastive learning approach for better clustering of similar fibers
  • State-of-the-art accuracy in segmenting challenging fiber tracts
  • Robust performance even with partial or noisy input data

Response Retrieval System using RAG

This project implements a sophisticated response retrieval system using the Retrieval-Augmented Generation (RAG) technique. By combining Large Language Models with efficient retrieval mechanisms, the system provides more accurate and contextually relevant responses.

Key Features:

  • Efficient document indexing and retrieval using vector databases
  • Integration with modern LLMs for generating contextually appropriate responses
  • BERT-based encoding for improved semantic understanding of queries
  • Customizable knowledge base for domain-specific applications
